# Remove Palindromic Subsequences

**Date:** 2026-07-20  
**Difficulty:** 🟢 **Easy**  
**Tags:** `Two Pointers` `String`

## Problem Description

You are given a string s consisting only of letters &#39;a&#39; and &#39;b&#39; . In a single step you can remove one palindromic subsequence from s . Return the minimum number of steps to make the given string empty . A string is a subsequence of a given string if it is generated by deleting some characters of a given string without changing its order. Note that a subsequence does not necessarily need to be contiguous. A string is called palindrome if is one that reads the same backward as well as forward. Example 1: Input: s = &quot;ababa&quot; Output: 1 Explanation: s is already a palindrom

[LeetCode Link](https://leetcode.com/problems/remove-palindromic-subsequences)

## Approach

Two Pointers

> Determine if the input string is a palindrome to decide the minimum steps to remove all characters

## Solution Logic

1. Initialize two pointers at the start and end of the string
2. Compare characters at the pointers and move them towards the center
3. If a mismatch is found, return 2 as the string can be removed in two steps

## Complexity Analysis

| Complexity | Analysis |
|------------|----------|
| **Time:** | O(n) — where n is the length of the string, as we potentially traverse the entire string once |
| **Space:** | O(1) — as we only use a constant amount of space to store the pointers and the count |

## Edge Cases Handled

- Empty string
- Single-character string
- Palindrome string

## What I Learned

The key insight is recognizing that if the string is not a palindrome, it can be made empty in two steps by removing all 'a's and then all 'b's

## Solution Code

```text
class Solution:
    def removePalindromeSub(self, s: str) -> int:
        i=0
        j=len(s)-1
        count=0
        while i<j:
            if s[i]!=s[j]:
                return 2
            i+=1
            j-=1
        if count==0:
             return 1
        return count
```
